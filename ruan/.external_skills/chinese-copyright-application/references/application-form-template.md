## 填写示例

- 软件全称：煤粉高温气流着火行为仿真与预测软件
- 版本号：V1.0
- 软件分类：应用软件
- 开发硬件环境：Intel Core i7 处理器，16GB 内存，512GB 固态硬盘，独立显卡
- 运行硬件环境：Intel Core i5 及以上处理器，8GB 内存，50GB 可用存储空间
- 开发操作系统：Windows 10 64 位
- 开发环境/工具：IntelliJ IDEA、VS Code、MySQL 8.0、Postman
- 运行平台/操作系统：Windows 10/11 64 位、Linux CentOS 7.x 及以上
- 运行支撑环境：JDK 17、MySQL 8.0、浏览器（Chrome/Edge）、Tomcat
- 编程语言：Java、HTML、CSS、JavaScript
- 源程序量：14309行
- 开发目的：实现煤粉高温气流着火行为仿真、参数预测与数据管理，支撑燃烧设备优化
- 面向领域/行业：能源电力、锅炉燃烧、热能动力、化工冶金、环保工程领域
- 技术特点类型：人工智能软件、大数据软件、云计算软件
