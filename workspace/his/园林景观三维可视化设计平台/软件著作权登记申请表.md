# 园林景观三维可视化设计平台 V1.0

## 一、软件基本信息
- 软件全称：园林景观三维可视化设计平台
- 版本号：V1.0
- 开发完成日期：2026-05-28
- 软件分类：应用软件（园林景观设计与评审管理）
- 编程语言：Python、JavaScript、HTML、CSS、JSON
- 运行环境：Windows（演示版为本地部署）

## 二、软件功能简介
本软件面向园林景观设计与项目评审场景，提供“场地基准信息管理、方案要素可视化表达、植物库与季相预览、材料字典展示、评审事项闭环、植物/材料/灯具清单导出”等能力。系统以浏览器为主要入口，强调方案的版本可追溯与评审过程可留痕，便于设计单位与甲方在沟通中对齐空间表达与清单口径。

## 三、主要功能模块
1) 账号与登录  
2) 场地与底图基准（演示口径）  
3) 方案编辑与季相切换（演示口径）  
4) 植物库维护与弹窗操作（新增/编辑/查看/删除）  
5) 材料字典查看  
6) 评审中心事项列表与状态口径  
7) 清单导出（植物/材料/灯具）  

## 四、软件技术特点（简述）
- 服务端：使用 Python 标准库 `http.server` 提供本地 HTTP 服务与登录接口，便于离线演示与材料归档。
- 前端：单页应用采用 Hash 路由组织“总览/场地/方案/植物/材料/评审/导出”视图，风格为低饱和绿与暖灰的卡片化布局。
- 数据：演示版内置示例数据与领域文本种子库，保证功能口径、截图与文档一致。
